# Weather App 🌦️

A weather application built using Async/Await, Fetch API and OpenWeatherMap API.

## Features
- Real-time weather data
- Temperature & humidity display
- Error handling for invalid cities

[Click here](https://preksha-weather-app.netlify.app)

## Tech Used
- HTML, CSS, JavaScript
- OpenWeatherMap API
- Async/Await, Fetch API
